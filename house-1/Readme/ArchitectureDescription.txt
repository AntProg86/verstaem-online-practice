
* App - инициализирующая логика приложения.
Глобальные стили приложения, локализация

* assets — глобальные статические assets, такие как изображения, 
SVG, логотип компании и т. д.

* components — глобальные общие/повторно используемые компоненты, 
такие как макет (оболочки, навигация), компоненты формы, кнопки

* store – глобальное хранилище Redux

* services — модули JavaScript

* utils — Утилиты, помощники, константы и т. д.

* pages - здесь будет содержаться большая часть приложения.

* data - файлы с данными

* types - глобальные общие/повторно используемые типы
